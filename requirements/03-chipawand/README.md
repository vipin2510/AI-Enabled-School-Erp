# Chipawand — School Folder

**ADESHWAR PUBLIC SCHOOL**
Chipawand, Chhattisgarh
*A Unit of Adeshwar Public School, Kondagaon, C.G.*

Chipawand's data is isolated from Kondagaon's and Pharasgaon's — students,
fees, attendance, library, results are all separate. Layer 2 (manager) and
Layer 1 (admin) accounts can switch between schools after login if they
have been granted access to multiple units.

Fill the CSVs in this folder in the order described in
